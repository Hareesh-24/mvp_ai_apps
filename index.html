About the Certificate System

The Smart Certificate Generator & Verification System is a web-based platform built using Django and integrated with the Ethereum blockchain. This system allows educational institutions to securely generate digital certificates for students or interns.

A unique cryptographic hash of each certificate is stored on the blockchain via a smart contract, ensuring the certificate is tamper-proof and immutable. Metadata such as student name, certificate ID, and issue date is saved locally using SQLite for fast admin access.

Every certificate is linked to a unique ID or QR code. During verification, the system rehashes the certificate and compares it with the blockchain-stored hash. If both match, the certificate is deemed authentic.

  • Frontend: React.js, HTML, CSS, JavaScript
  • Backend: Python (Django)
  • Blockchain: Ethereum for hash-based verification
  • Database: SQLite (locally stored metadata)
  • Verification: QR scan or ID input → hash comparison → validation
